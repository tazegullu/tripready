# TripReady Privacy Policy

This repository hosts the privacy policy page for the TripReady mobile app.

TripReady is a packing and travel checklist app designed to help users create trip checklists, packing lists, travel document reminders and before-you-go task lists.

## Privacy Policy

The privacy policy is available at:

https://tazegullu.github.io/tripready/

## About TripReady

TripReady does not require an account, does not use cloud sync and does not send trip or checklist data to an external server.

User-created trips, packing lists and checklist progress are stored locally on the user’s device.
